## Placeholders
<Callout title="Information" type="warn">
This require [PlaceholderAPI](https://www.spigotmc.org/resources/placeholderapi.6245/) to work! so make sure you have it installed!
</Callout>

| Placeholder                  | Description    |
| ----------------------------- | -------------  |
| %nextcollectors_current% | Shows how many chunk collectors that player has placed. |
| %nextcollectors_max%    | Shows how many chunk collectors that the player can place |
| %nextcollectors_auto_sell_summary% | Returns `true` or `false` depends on auto sell summary message status |
