## Placeholders
<Callout title="Information" type="warn">
This require [PlaceholderAPI](https://www.spigotmc.org/resources/placeholderapi.6245/) to work! so make sure you have it installed!
</Callout>

| Placeholder                  | Description    |
| ----------------------------- | -------------  |
| %nextgens_currentplaced% | Show the list of generators that players currently placed |
| %nextgens_max% | Show the maximum amount of generators that player can place |
| %nextgens_multiplier% | Show the sell multiplier that player has (Format: 1,250) |
| %nextgens_multiplier_short% | Same as above (Format: 1,25K) |
| %nextgens_event_name% | Shows the event name |
| %nextgens_event_time% | Shows the duration of the event or the incoming event |
| %nextgens_event_isactive% | Returns `true` or `false` |
| %nextgens_corrupt_time% | Shows time left before corruption event happen |
| %nextgens_total_generator% | Shows the total generator amount for the entire server |
| %nextgens_cashback% | Shows the cashback percentage that player has |
| %nextgens_statistics_gens_`<id>`% | Shows how many generators that player has for specific generators |
| %nextgens_statistics_earnings% | Shows total earnings (Format: 1,250,000) |
| %nextgens_statistics_earnings_formatted% | Shows total earnings (Format: 1,25M) |
| %nextgens_statistics_earnings_string% | Shows total earnings (Format: 1250000) |
| %nextgens_statistics_itemsold% | Shows total items that player have sold (Format: 1,250,000) |
| %nextgens_statistics_itemsold_formatted% | Shows total items that player have sold (Format: 1,25M) |
| %nextgens_statistics_totalsell% | Shows how many times player perform sell action (Format: 1,250,000) |
| %nextgens_statistics_totalsell_formatted% | Shows how many times player perform sell action (Format: 1,25M)
| %nextgens_statistics_sellwandsell% | Shows how many times player perform sell action using sellwand (Format: 1,250,000) |
| %nextgens_statistics_sellwandsell_formatted% | Shows how many times player perform sell action using sellwand (Format: 1,25M) |
| %nextgens_statistics_commandsell% | Shows how many times player perform sell action using sell command (Format: 1,250,000) |
| %nextgens_statistics_commandsell_formatted% | Shows how many times player perform sell action using sell command (Format: 1,25M) |
