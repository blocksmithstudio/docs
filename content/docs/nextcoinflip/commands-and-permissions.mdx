## Commands & Permissions

<Callout title="Information" type="info">
  [] = Optional<br/>
  &lt;&gt; = Required
</Callout>

| Command          | Description        | Permission |
| ---------------- | ------------------ | ---------- |
| coinflip | Open the coinflip main menu |  |
| coinflip [amount] | Wager using currency | |
| coinflip hand | Wager held item | |
| coinflip accept &lt;player&gt; | Accept an item coinflip challenge from other players | |
| coinflip reload | Reload the configurations | nextcoinflip.admin |
