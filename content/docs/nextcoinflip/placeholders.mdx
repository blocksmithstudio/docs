## Placeholders
<Callout title="Information" type="warn">
This require [PlaceholderAPI](https://www.spigotmc.org/resources/placeholderapi.6245/) to work! so make sure you have it installed!
</Callout>


| Placeholder                  | Description    |
| ----------------------------- | -------------  |
| %nextcoinflip_wins% | Shows the total wins statistics |
| %nextcoinflip_losses% | Shows the total losses statistics |
| %nextcoinflip_losestreak% | Shows the current lose streak statistics |
| %nextcoinflip_winrate% | Shows winrate statistics |
| %nextcoinflip_total_game% | Shows the total game that player's played |
| %nextcoinflip_earnings_&lt;currency&gt;% | Shows specific currency total earnings |
